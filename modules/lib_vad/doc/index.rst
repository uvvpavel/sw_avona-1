Voice Activity Detection Library
================================

Introduction
************

``lib_vad`` is a library providing an estimate of the probability of voice being present in an audio stream.

On GitHub
---------

``lib_vad`` is present as part of ``fwk_voice``. Get the latest version of ``fwk_voice`` from
``https://github.com/xmos/fwk_voice``. The ``lib_vad`` module can be found in the `modules/lib_vad` directory in ``fwk_voice``.
